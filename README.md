```

```

```
git clone https://github.com/ray0727/clean_robot.git
```

### Run robot_slave_base
```
roslaunch robot_navigation open_all_master.launch
```
### Run cooperation of two robot
```
roslaunch robot_navigation clean_robot.launch
```
